# SwearingWords

This repository contains a word list and extended word list of bad words in different languages.

## File Naming
```
words_<Language>(-extended).txt
```

## Languages
- de -> German
- en -> English

## Format
Each line contains a word. With a wait between 0 and 1,000,000
```
<Word>-<Wheight>
```
